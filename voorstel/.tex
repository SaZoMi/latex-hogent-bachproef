\begin{abstract}
Large Language Models (LLM's) demonstreren indrukwekkende capaciteiten in het genereren van code, maar het correct toepassen van complexe, domeinspecifieke APi's blijft een uitdaging. Game server managementplatformen zoals Takaro.io beschikken over uitgebreide API-endpoints die het genereren van correcte en herbruikbare codefragmenten belemmeren. Model Context Protocol (MCP) biedt een gestructureerde manier om de API-functionaliteiten op een gecontroleerde manier beschikbaar te maken voor LLM's. 
Dit onderzoek heeft als doel het ontwikkelen van een proof of concept (POC) van een MCP-ondersteunde coding agent die op basis van user prompts, correcte codefragmenten genereert voor game serverbeheer. De gegenereerde code kan na validatie door de gebruiker worden toegevoegd als herbruikbare module binnen het platform. Het onderzoek evalueert in welke mate MCP bijdraagt aan de correctheid van de gegenereerde code.
\end{abstract}
